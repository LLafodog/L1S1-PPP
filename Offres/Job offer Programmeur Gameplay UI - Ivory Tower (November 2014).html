Programmeur Gameplay UI

Town : Lyon (69)Job Description

Basé à Lyon et travaillant exclusivement avec UBISOFT, le studio de développement IVORY TOWER recherche un Programmeur Senior Gameplay (spécialité User Interface) pour travailler sur sa nouvelle IP de driving online The Crew.
Sous la responsabilité du Lead Programmeur UI, vous prendrez part au design et à la réalisation de modules de jeu ainsi que de leurs interfaces associées.

Candidate Profile

La passion du jeu vidéo, la créativité, la rigueur, l'autonomie, l'autocritique et l'esprit d'équipe seront indispensables, en s'appuyant sur les compétences suivantes :

Compétences requises :
  • Un minimum de 5 ans d'expérience en programmation de jeu
  • Maîtrise du C++
  • Expériences passées dans le développement d'UI de jeu
  • Connaissances approfondies de Flash et Action Script
  • Savoir travailler en étroite collaboration avec les game designers et les artistes UI
  • Forte sensibilité au game design
Compétences souhaitées :
  • Expériences passées dans le développement d'un moteur d'UI
  • Expériences passées sur le middleware ScaleForm ou autre middleware d'UI lié à Flash
  • Expériences passées sur un jeu multijoueur

Provide

CV, lettre de motivation, références, site Web (si applicable)
To: Responsable recrutement
